# fixit

fix roads

## Getting Started

### Dependencies

- Pip, python3

### How To Run

- Clone this repo.

- (Optional) Install dotenv with `pip install dotenv` and put your secret key there with SECRET as key. Example: `SECRET="YOURSECRETKEY"`

- Use the command `python manage.py runserver` to run
